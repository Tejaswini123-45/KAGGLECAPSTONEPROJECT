## Backup Plan

### If Something Breaks

**Gemini API Error**:
- "This is a rate limit issue. In production, we'd have fallback models."
- Show the generated files from previous runs

**Instagram Login Fails**:
- "Instagram sometimes requires verification. The post is generated and saved."
- Show the generated image and caption files

**Website Generation Slow**:
- "The AI is thinking hard! Usually takes 2-3 minutes."
- Show a previously generated website while waiting

**Network Issues**:
- Have screenshots/recordings ready
- Show the code and explain the architecture
- Walk through the generated output files
